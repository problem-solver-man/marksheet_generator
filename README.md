# Mark-Sheet Generator

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## Overview

This application is designed to generate mark sheets for school and college students efficiently and accurately. It significantly reduces manual effort, saves valuable time, ensures complete accuracy, and produces professionally formatted documents.

---

## Prerequisites

- Python 3.8 or higher
- Git

Ensure both are installed and available in your system PATH.

---

## Project Setup

Follow the steps below to set up the project:

1. Open Command Prompt (Windows) or Terminal (Linux/macOS).
2. Navigate to the directory where you want to clone the project.
3. Execute the following commands one by one.

### Windows

```bash
git clone https://github.com/problem-solver-man/marksheet_generator.git
cd marksheet_generator
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python src/app.py
```

### Linux & macOS

```bash
git clone https://github.com/problem-solver-man/marksheet_generator.git
cd marksheet_generator
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
python3 src/app.py
```

---

## Notes

- Replace the repository URL with your own if you are using a fork.
- If `python` refers to Python 2 on your system, use `python3` instead.
